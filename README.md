# 米哈游漫展ID卡图片生成器

还在因为去不了漫展/抢不到票而遗憾？还在被黄牛/代领的高价PUA？

该生成器让你直接获得图片进行自印

本程序部分代码逻辑参考[BHYG](https://github.com/ZianTT/BHYG)

## 支持游戏：

崩坏：星穹铁道/绝区零/原神


## 已有功能：

自由选择B站用户/B站玩家模板

选择官方提供的角色头像

修改左侧六行文字内容及颜色

修改左下角日期

自由开关右下角B站角标

将生成的图片用于现场打印（请注意使用该功能可能会导致被现场STAFF针对）
